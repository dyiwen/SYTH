
Tools: Code interface 常用代码复用整理
====

### Tools List:
---
    * RSYNC/
        * FTP_.PY
            * 同步ftp目录到本地、下载ftp文件到本地。
            * 依赖包: ftplib
    * EXT/
        * filter.py
            * DR,CT 文件过滤器
    * SYSTEM/
        * config_.py
            * 读写配置文件
        * logger_.py
            * 生成日志
        * os_.py
            * 创建目录-p ,删除空文件夹
        * threading_.py
            * 函数异步,线程池 ,对象池
    * DB/
        *DAL.py
            * Data Access Layer 数据库访问层
            * 依赖包: cx_Oracle MySQLdb
